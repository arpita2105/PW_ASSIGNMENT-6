How to add CSS ?
There are three ways to add CSS :-

  • INLINE CSS
  • INTERNAL CSS
  • EXTERNAL CSS

The following condition takes place when we target the same elements with inline, internal, and external CSS :-

  • The Inline CSS will have the highest priority and will override the internal and external CSS.
  • The Internal CSS will have the second highest priority and will override the external CSS but not the inline CSS.
  • The External CSS will have the lowest priority and will be overridden by the inline and internal CSS.

Priority Order

Inline CSS > Internal CSS > External CSS

Most specific CSS rule will be applied.
INLINE CSS is most specific.
